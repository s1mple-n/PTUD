
    - - - - - Hướng dẫn cài đặt các thành phần liên quan - - - - -

1. Chạy database trên SQL Server. Đường dẫn đến file mã nguồn cơ sở dữ liệu là:
database/QLBH_V16.sql

2. Cấu hình các thư viện liên quan:
- Thêm file jar dưới đây vào dependency của chương trình:
lib/LGoodDatePickerCustomizedByTN6KTPM15A2021.jar

3. Đường dẫn để đến class khởi động chương trình:
src/main/java/ui/BoKhoiDongChuongTrinh.java

- Tài khoản mặc định để đăng nhập:
+ Nếu thời điểm đăng nhập sau 08:00 và trước 16:00: 21100061 - 1111
+ Nếu thời điểm đăng nhập sau 16:00 và trước 22:00: 21100011 - 1111
